# Infrastructure as Code

Infrastructure automation for homelab using Terraform and Ansible.

## Structure

- `infra-terraform/` - Terraform configuration for Proxmox VM provisioning
- `config-ansible/` - Ansible playbooks and roles for Kubernetes cluster setup
- `Tools_config/` - Configuration guides and documentation

## Requirements

- Terraform
- Ansible
- HashiCorp Vault (for secrets management)
- Proxmox VE
